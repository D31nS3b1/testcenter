Das Institut zur Qualitätsentwicklung im Bildungswesen betreibt auf diesem Seiten eine Pilotanwendung für computerbasiertes Testen. Diese Web-Anwendung wurde entwickelt, um die Inhalte der Tests zu verwalten.

Um sich für die Verwaltung der Testinhalte anzumelden, geben Sie bitte den entsprechenden Namen und das Kennwort ein. Sie können über das Seitenmenü oben rechts ggf. weitere Funktionen aufrufen.

Derzeit sind Sie am System als Adminstrator/in angemeldet. Über das Menü oben rechts oder diesen Link gelangen Sie zur Startseite der Test-Verwaltung.

Bei Fragen zu diesem System wenden Sie sich bitte an Martin Mechtel.
