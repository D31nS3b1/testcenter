States of Running Tests

(explanatory text)

supervisor_account monitor_group
Tooltip: Testleiter
undefined

preview demo
Tooltip: Vorschau-Modus
undefined

person_outline pending
Tooltip: Test noch nicht gestartet
undefined

lock locked
Tooltip: Test gesperrt
Zustand kommt zu stande, wenn vom Teilnehmer der "Test beenden"-Button gedrückt wurde.

error error
Tooltip: Es ist ein Fehler aufgetreten!
Bei Teilnehmer-Rechner ist ein Fehler aufgetreten. Wahrscheinlich ein Netzwerkfehler oder ähnliches, kann aber auch auf Bugs hindeuten und sollte nach Möglichkeit untersucht werden. Der Teilnehmer sollte seinen Browser neu laden und probieren ob das Problem dann immer noch besteht.

sync_problem controller_terminated
Tooltip: Testausführung wurde beendet und kann wieder aufgenommen werden. Der Browser des Teilnehmers ist nicht verbunden und muss neu geladen werden!
Zustand kommt zustande, wenn der "Entsperren"-Knopf im gruppen-Monitor verwendet wurde.

error connection_lost
Tooltip: Seite wurde verlassen oder Browserfenster geschlossen!
Die Verbindung zum Teilnehmer-Browser ist abgerissen. Er könnte das Fenster geschlossen haben oder die Netzwerkverbindung ist abgerissen

pause paused
Tooltip: Test pausiert
undefined

warning focus_lost
Tooltip: Fenster/Tab wurde verlassen!
Ein anderes Fenster oder ein anderer Tab wurde angewählt, dei Seite ist jedoch immer noch offen und verbunden.

hourglass_full idle
Tooltip: Test ist 5 Minuten oder länger inaktiv!
undefined

play_circle_filled connection_websocket
Tooltip: Test läuft, Verbindung ist live
Test läuft und Teilnehmer ist im Live-Modus (= Websocket) verbunden.

play_circle_outline connection_polling
Tooltip: Test läuft
Test läuft und Teilnehmer ist im Polling-Modus Verbindung verbunden.

play_circle_filled ok
Tooltip: Test läuft
Test läuft, Verbindungstyp is unbekannt.
